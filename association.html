Abstract

  • Association Rule Learning is an approach that discovers the strength of relationships between different data points. It is commonly utilised to understand which products are frequently purchased together.
  • The ABC Grocery team wants to bring the alcohol department to life by bundling some promotions around the alcohol department and also help customers find relevant products nearby.
  • Association Rule Learning will help me understand which products should be placed near each other and what is the relationship between these products.

Introduction

The ABC Grocery Team wants to re-organize their alcohol department as there have been a lot of complaints from the customers lately. Complaints regarding finding products, finding relevant products, recommendations of the products that the customers must try.
The team is keen on bringing the alcohol department to life and also wants me to provide some strategies to bundle promotions around this department. My task is to provide some ideas to the team so that they can spark this department back to life and re-organize some proucts. Association Rule Learning helps in finding interesting and useful information that can be implemented in a business setting, for example bundeled promotions, buy product A get product B free or get a discount on B. It can also help grocery owners, stores of similar setting understand which products should be near each other in the store in order to make the journey of the customer as easy as possible. Therefore, I will be using Association Rule Learning to accomplish the task assigned.

Imlpementation

1. Reading the data

  • The ABC Grocery team has provided us with excel file that contain 3,500 alcohol transactions that contain 3,567 rows and 46 coulmns, i.e. there are 3,567 transactions and 45 products bought along with alcohol.
  • For reading the data I will use the Pandas library. Pandas has the read_csv functionality to read excel files that contain .csv extension.
  • I will convert the excel data into DataFrame alcohol_transaction as Pandas DataFrames are easy for Data Manipulation.

2. Data Preprocessing

  • Association Rule Learning can be implemented using the Apriori Algorithm and this algorithm does not accept data in the form of DataFrame or an Array. Instead it wants us to provide the data as a list of lists. Therefore, each transaction that I will have will be one list and this list will contain one element for each product in that transaction. I will implement these steps to achieve this:
  • Step 1 - Drop Transaction_ID column: Transaction_ID column is not needed in this implementation, hence I will drop the column.
    To achieve this - I will use the drop functionality of pandas to eliminate this column.
  • Step 2 - Creating a list of all Transactions: Creating a list of all transactions as Apriori algorithm does not accept DataFrames or Arrays.
    To achieve this - I will use the iterrows() to iterate through all the rows in the DataFrame. Drop the missing values using dropna function and use the append method to append every row to newly declared transaction_list.

3. Model Building

  • Step 1 - Import Apriori
    To import Apriori algorithm I will have to first install the package using pip install apyori.
    To bring in and use the functionality I will have use from apyori import apriori.
  • Step 2 - Build the Model
    I will pass the transaction_list that I created above into the Apriori Algorithm. The algorithm takes different parameters and thresholds that include the list, min_support, min_confidence, min_lif, min_lenght, max_length respectively.
    In the first parameter we pass in the data.
    min_support: Support is the percentage of all transactions that contain both item A and item B.
    min_confidence: Of all the transactions that include item A, what proportion also included item B.
    min_lift: The factor by which the confidence exceeds the Expected Confidence.
    min_length, max_length: How many combinations do I want to try.
