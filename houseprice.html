Overview

We are given a case study by the real estate company to predict the house prices that must be placed for their new project. They have provided us with different house prices based on different places. Attributes like crime rate, air quality, room numbers, age, distance have been provided with the historic prices. The company wants us to predict the prices for the new project.

Project Motivation

  • Predict the house prices based on different attributes provided.
  • Apply linear regression algorithm on the dataset and train the ML model.
  • Based on the trained model, test the new house prices.

Goal of this project

  • To predict house prices based on the data provided.
  • Apply Linear Regression Algorithm to train the model.
  • Test the algorithm on the new attributes to predict the prices.

Datasets

I obtained our data set usinng the following link: Data Link

  • The dataset contains 500 records of differnt houses with many attibures and also their prices that can be used to train the Machine Learning Model.
  • Based on the different attributes provided our task is to predict the new house prices. The final column to predict using the Linear Regression is the Price column.
