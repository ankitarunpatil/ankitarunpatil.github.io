Image Classification
Of Cats and Dogs

This project aims to to classify images for cats and dogs using different image classification models.
Predicting whether a image is cat or a dog based on different images provided.
Creating different Machine Learning models and training them to predict cat and dog images.

Overview

This case study deals with predicting images for cats and dogs. We have been provided with many images for cats and dogs. Our task is to take all of these images and build an ML model that will classify if an image is a cat or a dog. Machine Learning models like Keras and Tensorflow will be used to classify the images. We have been provided with 3000 images to train our ML models.

Project Motivation

  • To predict if an image is a cat or dog.
  • The solution to this problem is to create a Machine Learning model that will take an unseen image of a cat or a dog and classify them accordingly.

Goal of this project

  • To classify colored unseen cat or dog image.
  • Apply image classification techniques like Keras and Tensorflow to predict the images.
