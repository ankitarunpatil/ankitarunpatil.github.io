Classification
Oscar Winning Movies

This project aims to classify Oscar winning movies based on different attributes.
Classification algorithms like DecisionTrees, EGBoost, AdaBoost, GradientBoosting will be used to classify whether the movie will win an oscar or not.

Overview

In this case study, we have been provided with movies dataset, the training data contains different attributes of movies. The dataset also contains the movies that have won an Oscar or not. We have been tasked to predict whether a movie will win an Oscar or not based on the historic data provided to us. It is our task to create different Machine Learning Model that will predict if a movie will win an Oscar or not. For this dataset we have to give the best accuracy model that will predict Oscar Winning Movies in the future.

Project Motivation

  • Predicting Oscar Winning Movies based on the data provided.
  • Building different Machine Learning Models for predicting Oscar Winning Movies.
  • Figuring out the best model that will be used to predict the the Oscar Winning Movies.

Goal of this project

  • To classify Oscar Winning Movies.
  • Apply classification algorithms to the whole dataset to train different models.
  • Finding the best and accurate model by calculating model Accuracies.

Datasets

The dataset can be obtained using the following link: Data Link

  • The dataset contains 500 records that contain different movie attributes like MarketinExpense, ProductionExpense, Budget, MovieLength, Actor, DirectorRating, ProduerRating.
  • Based on the attributes our task is to predict the final column, Star_Tech_Oscar, i.e. based on all of the attributes if the movie is capable of winning an Oscar or not.
